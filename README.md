# 어머니의 손맛 레시피

> "이곳은 레시피가 아니라, 한 어머니의 시간이 머무는 공간입니다"

수십 년간 가족을 위해 차려온 밥상, 그 안에 담긴 사랑과 정성을 AI 기술로 되살리는 디지털 아카이브 프로젝트입니다.

## 주요 기능

- **요리 아카이브** - 어머니의 손때 묻은 레시피들을 계절별, 상황별로 분류
- **AI 아트 시네마** - AI가 상상한 어머니의 부엌, 음식이 완성되는 순간들
- **AI 갤러리** - 요리에 담긴 감성을 AI 아트로 표현
- **어머니 AI 톡** - 어머니의 관점에서 요리와 삶에 대해 대화
- **기록 & 비하인드** - 프로젝트의 비하인드 스토리

## 페이지 구성

| 페이지 | 파일 | 설명 |
|--------|------|------|
| 홈 | `index.html` | 메인 화면 |
| 소개 | `about.html` | 프로젝트 소개 |
| 요리 아카이브 | `recipes.html` | 레시피 목록 |
| 레시피 상세 | `recipe-detail.html` | 개별 레시피 |
| AI 시네마 | `cinema.html` | AI 영상 갤러리 |
| AI 갤러리 | `gallery.html` | AI 아트 작품 |
| 어머니 AI 톡 | `ai-talk.html` | AI 채팅 |
| 이야기 | `story.html` | 비하인드 스토리 |
| 연락하기 | `contact.html` | 연락처 |

## 기술 스택

- **HTML5** - 시맨틱 마크업
- **CSS3** - 반응형 디자인 (Flexbox, Grid)
- **JavaScript** - 인터랙션 (바닐라 JS)
- **Web Speech API** - AI 도슨트 음성 재생

## 실행 방법

별도의 서버 없이 `index.html` 파일을 브라우저에서 직접 열면 됩니다.

```bash
# 또는 로컬 서버 실행
npx serve .
```
